개선 및 업데이트 방향

  • ----------게시판-----------
  • map API 이용하여 캠핑 장소 모집 및 유저들과 캠핑 장비 리스트 통합
  • 게시물리스트 오름차순 내림차순 정렬
  • 중고거래 리스트 썸네일 이미지 띄우기
  • 중고 거래 완료 후, 캠핑 장비 수정 삭제 기능 없애기 또는 장비 삭제
  • ----------댓글-----------
  • 댓글 작성자 프로파일 이미지 보이기
  • 대 댓글 기능
  • ----------캠핑 장비-----------
  • 캠핑 장비 정렬
  • 유저가 등록한 캠핑 장비 검색 기능
  • 마이 기어 클릭시 상제 정보 모달창 띄우기
  • 중고 거래 작성시 장비 선택해야 작성 가능
  • ----------캠핑 참가 게시판-----------
  • 네이버 지도 API
  • 회원 참가 버튼 > 참가자 장비 리스트업
  • 참가자 장비 리스트 페이징
  • 참가자가 원하는 장비 선택해서 리스트업 하기
  • ----------회원-----------
  • 회원 프로파일 이미지 수정
  • 회원 이름 클릭시 회원 정보 보이기
  • 회원 가입시 아이디 중복 및 패스워드 size 제한
  • 로그인시 아이디 및 패스워드 오타 시 경고 메시지 띄우기
  • 쪽지 기능 (중고거래 연락)
  • 캠핑 모임 참가 리스트 캘리더에 띄우기
  • ----------보안-----------
  • JWT 적용
  • SSL 인증 (로드 밸런서 생성)
  • valid 검증
  • ----------css-----------
  • 첨부파일 업로드 로딩 표현
  • css link 처리
  • ----------중고거래-----------
  • 중고거래 등록된 캠핑 장비 수정 불가 처리
  • ----------기타-----------
  • 모바일 버전 반응형
  • CI/CD
  • bootstrap5 업그레이드
  • 잘못 업로드된 파일 삭제 [Quartz]

버그 및 오류

  • 중고거래 작성시 장비를 등록해야만 게시글 작성이 가능하게 구현하기
  • 캠핑 모임 게시글 수정시 약속 날짜 기본 벨류값을 데이터 값으로 로드하게 하기
